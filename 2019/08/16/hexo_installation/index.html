Hexo博客安装进行时

Posted on 2019-08-16 Edited on 2019-08-15

00 前言

换博客, 辣鸡wordpress

01 安装

本机安装git, Node.js
设置npm代理:

然后打开cmd 运行npm install -g hexo 安装hexo
新建一个文件夹，如MyBlog
进入该文件夹内，右击运行git，输入：hexo init（生成hexo模板, 从GitHub上download下来）

生成完模板，运行npm install
最后运行：hexo server （运行程序，访问本地localhost:4000可以看到博客已经搭建成功）
